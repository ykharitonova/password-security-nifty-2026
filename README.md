# password-security-nifty-2026
Repository for the [SIGCSE 2026 Nifty assignment](http://nifty.stanford.edu) files.

`assignment.md` contains the assignment instructions in Markdown. Original Word document as well as the start code can be found on the [assignment's Nifty page](http://nifty.stanford.edu/2026/bravo-kharitonova-password-security).

Feel free to contribute to this project by submitting a pull request ([here's how](https://docs.github.com/en/get-started/exploring-projects-on-github/contributing-to-a-project)).
